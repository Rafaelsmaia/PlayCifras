# PlayCifras 🎸

Uma plataforma moderna para busca e visualização de cifras, inspirada no Cifra Club.

## 🚀 Funcionalidades

- **Busca Inteligente**: Encontre cifras por música, artista ou gênero
- **Visualizador de Cifras**: Interface limpa e responsiva para ler cifras
- **Acordes Interativos**: Diagramas de acordes clicáveis
- **Sistema de Avaliação**: Avalie e veja as cifras mais populares
- **Filtros Avançados**: Filtre por dificuldade, gênero e popularidade
- **Design Responsivo**: Funciona perfeitamente em desktop e mobile

## 🛠️ Tecnologias

- **Next.js 14** - Framework React com App Router
- **TypeScript** - Tipagem estática
- **Tailwind CSS** - Estilização moderna
- **Lucide React** - Ícones
- **Framer Motion** - Animações

## 📦 Instalação

1. Clone o repositório:
```bash
git clone https://github.com/seu-usuario/playcifras.git
cd playcifras
```

2. Instale as dependências:
```bash
npm install
```

3. Execute o projeto:
```bash
npm run dev
```

4. Acesse [http://localhost:3000](http://localhost:3000)

## 🎯 Páginas Principais

- **Home** (`/`) - Página inicial com cifras populares
- **Buscar** (`/buscar`) - Sistema de busca com filtros
- **Cifra** (`/cifra/[id]`) - Visualizador de cifra individual

## 🎨 Design

O design foi inspirado no Cifra Club com:
- Cores principais: Verde (#00a651) e tons de cinza
- Layout limpo e focado na leitura
- Componentes reutilizáveis
- Animações suaves

## 🔧 Scripts Disponíveis

- `npm run dev` - Executa em modo desenvolvimento
- `npm run build` - Gera build de produção
- `npm run start` - Executa build de produção
- `npm run lint` - Executa linter

## 📱 Responsividade

O projeto é totalmente responsivo e funciona em:
- Desktop (1200px+)
- Tablet (768px - 1199px)
- Mobile (até 767px)
